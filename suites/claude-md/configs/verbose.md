# Project Instructions

## About this project
This is a JavaScript project. We care a lot about code quality and we want to
make sure that everything is done to a very high standard at all times. Please
take your time and think carefully about every change that you make.

## General guidance
When you are working on a task, please make sure that you fully understand the
requirements before you begin. Read the surrounding code carefully so that you
understand the existing patterns and conventions. Try to match the style of the
existing code as closely as you possibly can. If you are unsure about anything,
it is always better to ask than to guess.

## Coding standards
- Please write clean, readable, well-structured code.
- Please add comments where appropriate to explain what the code is doing.
- Please make sure that variable names are descriptive and meaningful.
- Please avoid introducing unnecessary complexity into the codebase.
- Please follow the existing conventions that you find in the project.
- Please make sure that your code is correct and handles edge cases.

## Testing
We have a strong testing culture on this project and we would really appreciate
it if you could make sure that all of the tests pass before you consider a task
to be complete. The tests can be run with the standard test command. Please do
not skip the tests, as they are very important to us.

## Final notes
Thank you so much for your help with this project. We really value your
contributions and we are grateful for the care and attention that you bring to
your work. Please always do your best.
